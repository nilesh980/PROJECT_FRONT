About Project

This project Aims to setup modernization of classroom by changing the way daily task are been done
• Attendance
• Notes Making

  • Attendance Management System

    A facial recognition-based attendance management system built on React would involve:
    1. Creating a UI that captures images of Each faces.
    2. Install React and necessary packages.
    3. Create a form for capturing images.
    4. Use a facial recognition API to detect faces.
    5. Compare detected faces to an existing database.
    6. Record attendance and display results.

  • Lives Notes making Through Speech

    Speech Recognition based model built using speech recognition framework and reactJS.
    1. Use speech-to-text software like Google Docs Voice Typing.
    2. Dictate notes in a clear and concise manner.
    3. Organize notes by topic or theme.
    4. Review and edit notes for accuracy and clarity.
    5. Save and share notes for easy access.

  • WEB-Fee-Image to Text

    Tesseract framework based model based on deep learning capable of converting web feed to text.
    1. Install Tesseract.js library in React project.
    2. Import Tesseract.js in React component.
    3. Define image and language options for Tesseract.
    4. Call Tesseract.recognize() function with image.
    5. Display extracted text in React component.

More About Frameorks Used

1. Face recognition using Flask KNN:
* Flask (Python web framework)
* K-Nearest Neighbors (KNN) algorithm (machine learning algorithm for classification)
* OpenCV (computer vision library for image processing)
* NumPy (Python library for scientific computing)
* Scikit-learn (Python library for machine learning)
2. Speech to text on React:
* React (JavaScript library for building user interfaces)
* Web Speech API (API for speech recognition in web browsers)
* JavaScript for handling API callbacks and rendering components
3. Text extraction from image using Tesseract:
* Tesseract.js (JavaScript library for optical character recognition)
* React (JavaScript library for building user interfaces)
* JavaScript for handling API callbacks and rendering components
